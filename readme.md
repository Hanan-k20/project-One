# Connect Four Game
![Connect Four Game puctier](<New folder/CONNECT FOUR 1.png>)

***This digital version of the classic Connect Four game was developed with a focus on an interactive user experience and aesthetic design.***

## Game Rules

1.  **Objective:** Be the first player to get **four of your colored discs in a row** (horizontally, vertically, or diagonally).
2.  **Players:** Two players only.
3.  **The Board:** A vertical grid with 6 rows and 7 columns.
4.  **Turns:** Players take turns dropping one disc at a time into any open slot at the top of a column.
5.  **Gravity:** Discs always fall to the lowest available space in the chosen column.
6.  **Winning:** The game ends immediately when a player completes a line of four.
7.  **Draw:** If the board is filled entirely before either player wins, the game is a draw.
